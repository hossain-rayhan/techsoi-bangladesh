1. Create a new sub-folder with a descriptive name (e.g., `healthcare-accessibility/`)
2. Add a `README.md` file inside the folder using the template below
3. Include any supporting documents (images, data files, research papers) in the same folder
4. Submit a Pull Request

## Issue Template

Create a `README.md` file in your issue folder with the following structure:

```markdown
# Issue Title

## Problem Statement
Describe the problem clearly and concisely.

## Current Situation
What is the current state? Include data/statistics if available.

## Impact
- Who is affected?
- How many people are affected?
- What are the consequences of not addressing this issue?

## Potential Areas for Tech Intervention
List possible technology-based approaches to address this issue.

## Supporting Documents
List any additional files included in this folder.

## Related Resources
Links to relevant articles, research papers, or existing initiatives.
```
