## How to Submit a Proposal

1. Create a new sub-folder with your project name (e.g., `digital-land-registry/`)
2. Add a `README.md` file inside the folder using the template below
3. Include all supporting documents (diagrams, mockups, technical specs) in the same folder
4. Submit a Pull Request

## Proposal Template

Create a `README.md` file in your proposal folder with the following structure:

```markdown
# Project Title

## Problem Statement
What problem does this project solve?

## Proposed Solution
Detailed description of your technology solution.

## Technical Architecture
- Technologies to be used
- System design overview
- Infrastructure requirements

(Include detailed diagrams in the `architecture/` folder)

## Implementation Plan
| Phase | Description | Timeline |
|-------|-------------|----------|
| Phase 1 | ... | ... |
| Phase 2 | ... | ... |

## Budget Estimate
Rough cost estimation for implementation.

(Include detailed breakdown in the `budget/` folder)

## Expected Impact
- Quantifiable benefits
- Number of people affected
- Transparency/anti-corruption impact

## Team Requirements
Skills and roles needed for implementation.

## Risks & Mitigation
Potential challenges and how to address them.

## Supporting Documents
List any additional files included in this folder.

## References
Related projects, research, or resources.
```
